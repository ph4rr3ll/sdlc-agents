# Qwen Code Integration

Qwen Code (Alibaba) — CLI-агент в духе Codex/Gemini Code. Поддерживает чтение `QWEN.md` или `AGENTS.md` как root-инструкцию. Изолированных субагентов нативно нет.

Режим работы: **File mode**. `STATE.json` и `handoffs/` — основной механизм.

## Структура

- `AGENTS.md` в корне (общий для нескольких инструментов, см. также `codex.md`).
- Опционально: `QWEN.md` — копия `AGENTS.md`, если Qwen Code в твоей версии не подхватывает `AGENTS.md`.

## Использование

Qwen Code при старте читает `AGENTS.md`. Тот содержит:
1. Краткое описание команды агентов.
2. Инструкцию вести себя как `manager` (т.е. при получении задачи следовать `.agents/manager.md`).
3. Обязательство работать через `.agent_space/[TASK_ID]/STATE.json` и `handoffs/`.

### Сценарий

```bash
qwen-code .
```

В чате:
```
Инициализируй задачу TASK-001: добавь экспорт отчётов в CSV
```

Qwen Code в режиме менеджера:
1. Создаёт `.agent_space/TASK-001/STATE.json`.
2. Пишет бриф для `shaper` в `WORKSPACE/handoffs/01_shaper_brief.md`.
3. Обновляет `STATE.json` (`active_agent: shaper`, `current_stage: SHAPING`).
4. Сообщает: "Бриф для shaper готов. Запусти специалиста, прочитав `.agents/shaper.md` и `WORKSPACE/handoffs/01_shaper_brief.md`."

### Переключение «ролей» в одном инструменте

Так как Qwen Code держит один контекст, переключение на специалиста делается **новой сессией** или явным prompt-инструктированием:

```
Начни новую задачу. Ты — shaper.
Прочитай свой манифест в `.agents/shaper.md`, затем
прочитай свой бриф в `.agent_space/TASK-001/handoffs/01_analyst_brief.md`
и действуй строго по ним.
```

Это эмулирует субагент-изоляцию: явный prompt-reset + чтение манифеста + чтение брифа.

## Ограничения

- Нет реальной изоляции контекста. История диалога копится, и через несколько ролей контекст становится зашумлённым. **Рекомендация: начинать новую сессию Qwen Code на каждого специалиста.**
- Нет параллелизма.
- Tool restrictions per-role — не поддерживаются. Полагайся на дисциплину агента (CONSTRAINTS в манифесте).

## Минимальный AGENTS.md для Qwen Code

См. `AGENTS.md` в корне репозитория — он спроектирован совместимым.
